We solve
\[
y'' + 9y =
\begin{cases}
\cos t, & 0 \le t < \tfrac{3\pi}{2},\\
\sin t, & t \ge \tfrac{3\pi}{2},
\end{cases}
\quad y(0)=0,\;y'(0)=0.
\]

The solution is
$$
y(t) = 
\begin{cases}
\displaystyle
\frac{1}{8}\bigl(\cos t - \cos(3t)\bigr),
& 0 \le t < \tfrac{3\pi}{2},\\[1em]
\displaystyle
-\frac{1}{6}\cos(3t)\;+\;\frac{1}{8}\sin(3t)\;+\;\frac{1}{8}\sin t,
& t \ge \tfrac{3\pi}{2}.
\end{cases}
$$
